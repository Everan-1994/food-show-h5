文件说明：

html文件说明：

    aboutUs.html 关于我们文件
    brand.html 品牌文件
    brandCooperation.html 品牌合作文件
    brandDetail.html 品牌详情
    contactUs.html 联系我们
    contactUsForm.html 联系我们form 表单
    cooperation.html 商超合作
    cooperationDetail.html 商超合作详情
    index.html 首页
    search.html 搜索页面


css 文件说明：
    html文件名和css文件名一致

    额外说明：
        accordion.css 为折叠菜单样式
        certify.css 为首页轮播图样式
        common.css 为抽取公共样式
        reset.css 为重置样式
        swiper.min.css 为轮播插件样式

js 文件说明：
    accordion.js 为折叠菜单js代码
    menu.js 为手机侧边栏滑动导航
    swiper.min.js 为swiper 轮播插件代码
    tabs.js 为tab选项卡切换js代码
    video.js 为视频截图js代码
